# 🌐 Aditya Sidham's Personal Portfolio

Welcome to my personal portfolio website! This site is a reflection of my journey as a Data Science graduate student at the University of Southern California (USC), showcasing my technical skills, achievements, and projects in software development, machine learning, and IoT.

## 🚀 Live Demo
🔗 [View the Live Site](https://adityasidham.github.io/AdityaPortfolio/)

---

## 📌 Overview

This responsive portfolio website is built using the **DevFolio Bootstrap template**, customized with my own content, animations, and dark mode support. It includes:

- 🧑‍💼 **About Me** section with background and interests
- 🛠 **Projects & Internships** with visuals and detailed descriptions
- 🏆 **Noteworthy Highlights** like certifications, patents, and achievements
- 🎓 **Academic Timeline** tracking my education and professional growth
- 📬 **Contact Form** integrated with Formspree
- 🌙 **Dark Mode** toggle for better viewing experience

---

## 🔧 Tech Stack

- **HTML5**
- **CSS3** (Custom styling and Bootstrap)
- **JavaScript** (Dark mode, AOS animations)
- **Bootstrap 5.3**
- **Font Awesome + Bootstrap Icons**
- **GitHub Pages** (for deployment)
